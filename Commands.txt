Create Vite Project:

npm create vite@latest
   |
   | React
      |
      | Javascript

Open project folder and open new terminal and install tailwindcss:

npm install tailwindcss @tailwindcss/vite


Now go to vite.config.ts and import tailwindcss :

import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react() , tailwindcss()],
})

Now go to index.css remove all te conent in ther and import :

@import "tailwindcss";
